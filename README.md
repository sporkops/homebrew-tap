# homebrew-tap

Official Homebrew tap for [Spork](https://sporkops.com).

## Installation

```sh
brew tap sporkops/tap
brew install spork
```

## Upgrade

```sh
brew update
brew upgrade spork
```

## Usage

```sh
spork --version
spork --help
```

## Manual tap setup

If you prefer to reference the tap explicitly:

```sh
brew install sporkops/tap/spork
```
